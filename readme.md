# GBNet #

Rust library for multiplayer game networking.

## Features ##

- Connections
- Reliable Messages
- Large Data Transfer
- Packet Fragmentation
- Bit-Packed Serialization
- Packet Delivery
- State Synchronization
- Snapshot Compression
- Snapshot Interpolation
- Deterministic Lockstep
- Congestion Avoidance
- Fixed Timestep
